# Data Science Learning Repository

Welcome to my Data Science learning repository! Here, I'm documenting my learning journey from scratch, covering various technologies, libraries, and concepts related to Data Science.

## Overview

This repository contains my notes, code, and resources I've gathered while learning Data Science from multiple sources including YouTube channels like CampusX (Nitish Singh), Krish Naik, and websites like GeeksforGeeks, Coursera, Towards Data Science, Analytics Vidhya, etc.

### Technologies & Libraries Covered

1. **Python**: Fundamental programming language for data manipulation and analysis.
2. **OOPs**: Object-Oriented Programming concepts in Python.
3. **Numpy**: Library for numerical computing in Python.
4. **Pandas**: Library for data manipulation and analysis.
5. **Matplotlib**: Visualization library for creating static, interactive, and animated visualizations.
6. **Seaborn**: Statistical data visualization based on Matplotlib.
7. **Scikit Learn**: Machine learning library for classical ML algorithms.
8. **Tensorflow**: Deep learning framework for building neural networks.
9. **SQL**: Query language for managing and manipulating databases.
10. **Flask**: Web framework in Python for building web applications.
11. **Natural Language Processing**: Techniques for working with human language data.
12. **Deep Learning**: Advanced techniques in neural networks for complex problem-solving.
13. **Machine Learning**: The study of algorithms and statistical models for learning patterns from data.
14. **Statistics**: Understanding statistical concepts and their applications in data analysis.
